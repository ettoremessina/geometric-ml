# geometric-ml

A series of experiments in geometric machine learning on 3D point clouds, with a focus on fiber bundles and manifold-aware representations.

## Experiments

### [bundlenet01](./bundlenet01)

Procedural point-cloud dataset generator, desktop viewer, and PointNet classifier. The first experiment in the series: it introduces surface normals as tangent bundle features and measures their effect on classification accuracy.

10 object classes · 2048 points per cloud · XYZ + surface normals · PointNet with T-Net alignment
